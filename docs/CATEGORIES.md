# GoAgent 文档分类

## 文档目录结构

```
docs/
├── guides/                    # 用户指南
│   ├── quick_start.md        # 快速开始
│   └── faq.md                # 常见问题
├── development/              # 开发指南
│   ├── integration.md       # 集成指南
│   ├── testing.md           # 测试指南
│   └── performance.md       # 性能调优
├── architecture/             # 系统设计
│   ├── arch.md              # 架构设计
│   └── retrieval-strategy.md # 检索策略
├── features/                 # 功能模块
│   ├── experience-system.md # 经验系统
│   ├── memory-distillation.md # 记忆蒸馏
│   └── llm_query_rewrite.md # LLM 查询重写
└── components/               # 组件文档
    ├── memory/              # 记忆组件
    ├── llm/                 # LLM 组件
    └── storage/             # 存储组件
```

## 分类说明

### 📚 用户指南
面向新用户的基础文档，帮助快速上手。

### 🔧 开发指南  
面向开发者的深入文档，包括集成、测试和性能优化。

### 🏗️ 系统设计
架构设计文档，包括系统架构、数据流等。

### ⚡ 功能模块
特定功能的详细文档，如经验系统、记忆蒸馏等。

### 🧩 组件文档
各个组件的详细技术文档。
